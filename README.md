### Installation
1. This project uses the [uv](https://docs.astral.sh/uv/guides/install-python/) package manager  
`pip install uv`  

2. Clone the repo  
`git clone https://github.com/ayushjay/pomoPy.git `  
`cd myproject`

3. Install the dependencies and run  
`uv sync`  
`python3 manage.py runsever`
